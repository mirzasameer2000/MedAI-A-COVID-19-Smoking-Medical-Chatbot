# MedAI: A COVID-19 & Smoking Medical Chatbot

This is a ChatGPT-style medical assistant built in Google Colab using LlamaIndex, Hugging Face models, and Gradio.

## Features
- Answers medical questions based on the CORD-19 dataset
- Uses semantic search + lightweight open-source LLM
- Chat interface built with Gradio

## Run on Colab
Open this notebook in [Google Colab](https://colab.research.google.com/) and run each cell in order.

## Requirements
See `requirements.txt`.
